## 🗺️ Learning Order (recommended)

### Path A: Data Analyst
```
SQL → Excel/Sheets → Python → Power BI/Tableau → Git (basic)
```

### Path B: BI Developer
```
SQL → Power BI → Data Modeling → DAX → Git → Python (light)
```

### Path C: Analytics Engineer / BI Engineer
```
SQL → Git → Python → dbt → Airflow → Redshift/Snowflake → Docker → Terraform
```

### Path D: Data Governance
```
SQL → OpenMetadata → Python (scripting) → Git → RBAC concepts
```
